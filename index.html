Web Developement Knowledge

HTML(Hyper Text Markup Language)

Hyper text means links to internal or external soruce like ads links( external link) or one page to another example, Home page to About page(internal link). While Markup means set of Instruction to be executed by Web Browser like Google Chrome, Mozila Firefox,etc. It is basically is a markup or set of instructions that defines how content is display on Web page.

CSS (Cascading Style Sheet)

It is basically a document like HTML which describe how the content will be presented or their layout. It contains set of design or style rule on basis of which the content is presented. Basically CSS concern with the Presentation or Layout of content in Web Page.

JavaScript(Programing Language)

It is basically only Programming Language that Web Browser understands. It looks after interactivity of the Web page and content inside it. That is, it concern with the clicking of buttons and other such events.

Conclusion

We can say, HTML is like skeleton of the web page which defines the content of web page whereas CSS is like muscle or skin that is layout or style in which content to be display. And the last JavaScript is like function of body that is how content will interact with user.
